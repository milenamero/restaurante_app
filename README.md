## Descripción
Este proyecto es una aplicación modular en Python para gestionar clientes y productos de un restaurante, aplicando principios de Programación Orientada a Objetos (POO).

## Estructura del proyecto
- `modelos/`: Contiene las clases entidad (`Producto`, `Cliente`).
- `servicios/`: Contiene la clase `Restaurante` con la lógica de negocio.
- `main.py`: Punto de entrada de la aplicación.

## Cómo ejecutar el proyecto
1. Asegúrate de tener instalado Python en tu computadora.
2. Abre una terminal en la carpeta raíz del proyecto.
3. Ejecuta el siguiente comando:
   `py main.py`

## Reflexión sobre la Modularidad
La modularidad es fundamental en el desarrollo de software. Al separar las responsabilidades en diferentes archivos (modelos y servicios), logramos un código más limpio, fácil de mantener y escalar. Esto evita que el sistema sea un "bloque" difícil de modificar y permite trabajar de forma más ordenada.
